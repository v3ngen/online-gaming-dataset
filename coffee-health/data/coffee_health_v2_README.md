# COFFEE HEALTH DATASET (v2)

This is a dataset for understanding the factors that contribute to a person's health, in particular coffee consumption, lifestyle, and self-rated health (assignment 1) and to run machine learning experiments to determine if it's possible to accurately make classifications (assignment 2).

There is one feature you should use for classification for assignment 2, namely:
- **SelfRatedHealth**: Poor / Fair / Good / Very Good / Excellent

The dataset contains a mixture of numerical and categorical features, some of which include noisy, inconsistent, or missing values to reflect real-life data challenges. This requires you to demonstrate your abilities with data analysis of data quality and handling any issues you encounter so as to a) perform an accurate exploratory data analysis (assignment 1), and b) perform appropriate pre-processing required for applied machine learning (assignment 2).

Including the class column, the dataset has got 16 features:
- **ID**: A unique ID for each person record (row)
- **Age**: Age of the person in whole years
- **Gender**: Male / Female / Other
- **Country**: Country of residence (Italy / France / UK / Norway)
- **Daily Coffees**: Measured in cups
- **Caffeine Intake**: In mg
- **Avg Sleep Hours Per Night**: A typical/average value, self-reported or from a sleep-tracking wearable
- **Sleep Quality**: A typical/average quality, from a wearable or self-assessed as Poor / Fair / Good / Excellent
- **BMI**: Body Mass Index
- **Avg Resting Heart Rate**: A typical/average value, from a wearable or clinical measurement
- **Physical Activity Level**: Sedentary / Lightly Active / Moderately Active / Very Active
- **Health Issues**: No Issues / Mild / Moderate / Severe
- **Smoking Status**: Never / Former / Light Smoker / Heavy Smoker
- **Alcohol Level**: Non-Drinker / Light / Moderate / Heavy
- **Stress Level**: Low / Medium / High — self-evaluated or from an activity tracker
- **SelfRatedHealth**: Poor / Fair / Good / Very Good / Excellent (the class column)

OBS: This dataset is based on a dataset available online. However, it was completely re-generated, and some features have changed. You MUST use this version, otherwise you will be subject to getting zero marks.
